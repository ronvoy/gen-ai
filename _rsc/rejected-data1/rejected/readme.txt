------------------ LAMBADA (PARTIAL) REJECTED DATA ------------------

This archive contains part of the rejected data of the LAMBADA dataset
(LAnguage Modeling Broadened to Account for Discourse Aspects). In
particular, it includes passages selected based on the low probability
that the language model assigned to the correct answer, and that
participants could guess by considering the last sentence in the
passage only, without looking at the broader context (the LAMBADA set
proper contains thoese cases where subjects needed the broader
context).

Information on LAMBADA and the rejected data can be found in the
following paper -- PLEASE CITE IT IF YOU USE LAMBADA OR THE REJECTED
DATA IN PUBLISHED WORK (you will find the bibtex entry below):

D. Paperno, G. Kruszewski, A.  Lazaridou, Q. Pham, R. Bernardi,
S. Pezzelle, M. Baroni, G. Boleda and R. Fernandez. 2016. The LAMBADA
dataset: Word prediction requiring a broad discourse
context. Proceedings of ACL 2016 (54th Annual Meeting of the
Association for Computational Linguistics), East Stroudsburg PA: ACL,
pages 1525-1534.

You will find 1 file besides this readme in the archive:

rejected_plain_text.txt

This text file contains data that was rejected during our data
collection process.

---NOTE: In this file each line corresponds to a passage, including
target sentence and target word. For each passage, the word to be
guessed is the last one.

---NOTE2: The source data for this dataset as well as LAMBADA come
from the Book Corpus, made in turn of unpublished novels (see Y. Zhu,
R. Kiros, R.f Zemel, R. Salakhutdinov, R.  Urtasun, A. Torralba and
S. Fidler. Aligning books and movies: Towards story-like visual
explanations by watching movies and reading books. ICCV 2015, pages
19-27).
